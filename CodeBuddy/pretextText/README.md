# Pretext Text MVP

Two standalone demos using [@chenglou/pretext](https://github.com/chenglou/pretext).

- [index.html](./index.html) — hub
- [auto-fit.html](./auto-fit.html) — HTML layout, Pretext computes font size only
- [noodle-drag.html](./noodle-drag.html) — soft-body text drag on Canvas
