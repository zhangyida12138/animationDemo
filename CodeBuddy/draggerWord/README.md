# Animation Notes

On load, card content drops from above. Drag words with the mouse. This demo uses [matter.js](https://brm.io/matter-js/) for physics.

## Reference

Inspired by the left panel animation on Tencent CodeBuddy: https://www.codebuddy.ai/home

## Tips

You can remove the left/right walls and tweak typography. Like CodeBuddy, start with gravity at `0` and enable gravity on hover.
