# Animation Demo

This project reproduces and practices classic interactive animations from major tech products.

## Tech Stack

- **HTML** — page structure and semantics
- **CSS** — animations, transitions, and layout
- **JavaScript** — interaction logic and dynamic control

No React, Vue, or other frameworks; animation libraries are avoided when possible. Implementations use native Web APIs so you can study how motion works under the hood.

## Goals

- Collect and recreate motion design worth learning from products at companies such as Apple, Google, Meta, and ByteDance
- Break down keyframes, easing curves, and state changes with plain frontend code
- Serve as a reference repo for motion design study

## Usage

Open each example’s `index.html` in a browser. No build step or dependencies required.

## Layout

Each effect lives in its own folder with `index.html` and optional `style.css` / `script.js`.

```
animationDemo/
├── README.md
└── examples/          # motion demos (add as needed)
    └── example-name/
        ├── index.html
        ├── style.css
        └── script.js
        └── README.md
```
