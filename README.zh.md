# Animation Demo

本项目用于复现和练习各大厂产品中的经典交互动画效果。

## 技术栈

- **HTML** — 页面结构与语义
- **CSS** — 动画、过渡与布局
- **JavaScript** — 交互逻辑与动态控制

不依赖 React、Vue 等框架，尽可能也不使用动画库，尽量全部使用原生 Web 技术实现，便于理解动画原理与底层实现。

## 项目目标

- 收集并还原大厂（如 Apple、Google、Meta、字节等）产品中值得学习的动效
- 通过纯前端代码拆解动效的关键帧、缓动曲线与状态切换
- 作为动效学习与参考的代码仓库

## 使用方式

直接在浏览器中打开对应示例的 HTML 文件即可预览效果，无需构建或安装依赖。

## 目录说明

每个动效示例单独放在一个文件夹中，包含 `index.html` 及相关的 `style.css`、`script.js`（可按需拆分）。

```
animationDemo/
├── README.md
└── examples/          # 各动效示例（按需添加）
    └── example-name/
        ├── index.html
        ├── style.css
        └── script.js
        └── README.md
```
